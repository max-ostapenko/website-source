# website-source

## Initial setup

```bash
# Install Deno
curl -fsSL https://deno.land/x/install/install.sh | sh

# Install Lume
deno run -Ar https://deno.land/x/lume/init.ts

# Firebase Tools
npm install -g firebase-tools
```
